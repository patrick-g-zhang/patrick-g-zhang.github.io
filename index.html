Examples for prosody control at syllable level

1. The syllables whose prosody are changed are marked with red color
2. The code 3 is related to prominence phenomenon of syllable while code 6 is related to reduced phenomenon of syllable.

1.
Text: I do not know to you a little .
Syllables: AY1 | D+UW1 | N+AA1+T | N+OW1 | W+AH1+T | IH1+T | IH1+Z | AO1+L | AX0 | B+AW1+T

a.
prosody code of syllable 'N+OW1'(know) is placed with code 3


b.
prosody code of syllable 'N+OW1'(know) is placed with code 6


c.
prosody code of syllable 'N+OW1'(know) is placed with code 5
